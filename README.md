# Portfolio Website

A sleek, mobile-friendly personal profile website designed to showcase your social links, projects, and contact information in one simple page.

---

## Features

- Mobile-first design – optimized for phones and tablets
- Fast and lightweight – minimal and responsive layout
- Social links hub – easy access to all platforms
- Project showcase – highlight development work
- Contact section – quick ways to get in touch
- Clean modern UI – simple, sleek aesthetic

---

## Preview

The site functions as a central profile page where visitors can quickly find:

- Social media profiles
- Development projects
- Contact information
- Personal branding

Think of it as a minimal portfolio + link hub.
